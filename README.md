# Particle Halo - 粒子光环动画

一个基于 React Three Fiber 和 GLSL 着色器的交互式粒子动画效果。

## 技术栈

| 技术 | 版本 | 用途 |
|-----|------|------|
| React | 19.2.0 | UI框架 |
| Three.js | 0.182.0 | 3D渲染引擎 |
| React Three Fiber | 9.5.0 | React的Three.js封装 |
| Vite | 7.0.0 | 构建工具 |
| GLSL | - | 顶点/片段着色器 |

## 核心实现思路

### 1. 粒子网格系统

- **数量**: 5500个粒子 (100列 × 55行)
- **布局**: 网格宽度40，高度22
- **随机抖动**: 每个粒子位置有 ±0.25 的随机偏移，避免过于整齐
- **渲染方式**: 使用 `InstancedMesh` 实现高性能渲染

### 2. 五层动画系统

#### 第1层: 基础流动
```
漂移速度 = 时间 × 0.15
X位移 = (sin(漂移速度 + y×0.5) + sin(漂移速度×0.5 + y×2)) × 0.25
Y位移 = (cos(漂移速度 + x×0.5) + cos(漂移速度×0.5 + x×2)) × 0.25
```
让粒子像水流一样缓慢漂移。

#### 第2层: 水母光环
- 围绕鼠标形成椭圆光环 (X缩放1.3, Y缩放1)
- 光环半径动态变化: `2.4 + sin(时间×0.8) × 0.5`
- 边缘粒子被推开，产生呼吸效果
- 添加噪声让边缘不规则

#### 第3层: 外层振荡
- 光环外围粒子产生振荡
- 频率: 2.6，幅度: 0.76
- 生效范围: 光环半径+0.4 到 光环半径+2.2

#### 第4层: 大小变化
- 基础大小: 0.016
- 激活大小: 0.044 (在光环边缘)
- 粒子形状: 椭圆 (X:1, Y:0.6)

#### 第5层: 旋转抖动
- 基础旋转速度: 0.1
- 抖动幅度: 0.2
- 每个粒子有随机相位

### 3. 鼠标交互系统

```javascript
// 拖拽效果 - 非常慢的跟随
dragFactor = 0.015

// 抖动效果 - 让鼠标位置有自然抖动
jitterX = (sin(时间×0.35) + sin(时间×0.77+1.2)) × 0.5
jitterY = (cos(时间×0.31) + sin(时间×0.63+2.4)) × 0.5
```

### 4. 颜色系统

四色动态混合:
- 基础色: `#0000ff` (深蓝)
- 颜色1: `#4285f5` (浅蓝)
- 颜色2: `#eb4236` (红色)
- 颜色3: `#faba03` (黄色)

颜色根据位置和时间动态混合，产生流动的色彩效果。

### 5. 粒子形状

使用 **Superellipse** 公式绘制粒子:
```
d = pow(pow(|x|, 2.6) + pow(|y|, 2.6), 1/2.6)
```
产生介于圆形和方形之间的有机形状。

## 运行方式

```bash
cd particle-halo
npm install
npm run dev
```

访问 http://localhost:5200

## 性能优化

- 使用 `InstancedMesh` 批量渲染
- 着色器计算在 GPU 执行
- 使用 `useMemo` 缓存几何体和材质
- 透明粒子使用 `depthWrite: false` 避免深度冲突
